# Marcus Silva | Engenharia de Software

Landing page empresarial apresentando serviços de arquitetura de software, desenvolvimento full stack e consultoria técnica.

## 🎯 Sobre

Site profissional que apresenta a expertise em:
- **Arquitetura de Software** - Design escalável e sustentável
- **Desenvolvimento Full Stack** - Soluções completas e integradas
- **APIs e Integrações** - Comunicação entre sistemas
- **Consultoria Técnica** - Orientação estratégica para produtos digitais

## ✨ Características

- **Design Moderno**: Interface dark mode elegante e profissional
- **Responsivo**: Totalmente adaptável para dispositivos móveis
- **Performance**: Carregamento rápido com CSS otimizado
- **Acessibilidade**: Navegação clara e intuitiva
- **SEO Otimizado**: Meta tags descritivas e estrutura semântica

## 🏗️ Estrutura

```
LP-empresarial/
├── index.html          # Arquivo principal com HTML, CSS e estrutura
└── README.md          # Este arquivo
```

### Seções do Site

1. **Header** - Navegação e branding
2. **Hero Section** - Apresentação principal com CTA
3. **Serviços** - Card com principais ofertas
4. **Stack Tecnológico** - Tecnologias utilizadas
5. **Resultados** - Casos de sucesso e métricas
6. **Sobre** - Apresentação pessoal
7. **Contato** - Formulário de comunicação
8. **Footer** - Links e informações adicionais

## 🛠️ Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização avançada com variáveis CSS
- **Tailwind CSS** - Framework utilitário para design responsivo
- **JavaScript** - Interatividade (scroll suave)

## 📱 Responsividade

O site é totalmente responsivo com breakpoints:
- **Desktop**: Versão completa com navegação horizontal
- **Mobile** (< 768px): Menu reduzido, layout otimizado

## 🚀 Como Usar

1. Abra o arquivo `index.html` em um navegador web
2. Navegue pelas seções usando o menu ou scroll
3. Utilize os botões de CTA para chamar à ação
4. Preencha o formulário de contato para enviar mensagens

## 💻 Customizações Possíveis

- Editar as seções de conteúdo diretamente no HTML
- Ajustar cores alterando as variáveis CSS no `:root`
- Modificar espaçamento e layout com classes Tailwind
- Integrar formulário de contato com backend
